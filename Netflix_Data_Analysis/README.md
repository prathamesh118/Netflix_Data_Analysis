# 🎬 Netflix Data Analysis

A Data Cleaning and Visualization project that explores the Netflix Movies and TV Shows dataset using Python. The project focuses on preprocessing raw data, performing exploratory data analysis (EDA), and generating visual insights to better understand Netflix's content library.

---

## 📌 Project Overview

This project demonstrates the complete data analysis workflow:

- Importing and exploring raw data
- Cleaning missing and inconsistent values
- Removing duplicate records
- Handling outliers (where applicable)
- Performing Exploratory Data Analysis (EDA)
- Creating informative visualizations
- Extracting meaningful insights

## 📊 Dataset

The dataset contains information about Netflix Movies and TV Shows, including:

- Show ID
- Title
- Type (Movie / TV Show)
- Director
- Cast
- Country
- Date Added
- Release Year
- Rating
- Duration
- Genre
- Description

---

## 🧹 Data Cleaning

The following preprocessing steps were performed:

- Removed duplicate records
- Handled missing values
- Converted date columns into datetime format
- Standardized text formatting
- Checked for inconsistent values
- Prepared the dataset for visualization

## 📈 Visualizations

The project includes visualizations such as:

- Content Type Distribution
- Movies vs TV Shows
- Top 10 Countries Producing Netflix Content
- Ratings Distribution
- Release Year Distribution
- Most Popular Genres
- Correlation Heatmap
- Missing Value Analysis

---

## 🔍 Key Insights

- Movies significantly outnumber TV Shows on Netflix.
- The United States contributes the highest amount of content.
- Most Netflix titles were released after 2015.
- TV-MA is among the most common content ratings.
- Drama and International Movies are highly represented genres.
